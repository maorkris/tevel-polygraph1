## 🚀 התקנה והפעלה

### אופציה 1: GitHub Pages (חינם!)

1. העלה את הפרויקט ל-GitHub
2. לך ל-Settings > Pages
3. בחר את ה-branch `main` ותיקיית `/ (root)`
4. לחץ Save
5. האתר יהיה זמין ב: `https://YOUR_USERNAME.github.io/tevel-polygraph-website`

### אופציה 2: שרת אחסון רגיל

פשוט העלה את כל הקבצים לתיקיית ה-public_html בשרת שלך.

## ⚙️ הגדרות נדרשות

### הגדרת טופס יצירת קשר (FormSubmit)

1. פתח את `index.html` ו-`contact.html`
2. מצא את השורה:
   ```html
   action="https://formsubmit.co/YOUR_EMAIL@DOMAIN.COM"
   ```
3. החלף `YOUR_EMAIL@DOMAIN.COM` בכתובת המייל שלך
4. בפעם הראשונה שמישהו ישלח טופס, תקבל מייל אישור מ-FormSubmit

### הוספת תמונות

יש להוסיף את התמונות הבאות לתיקיית `images/`:
- `logo.png` - לוגו החברה
- `favicon.png` - אייקון לטאב
- תמונות למאמרים ולקוחות
